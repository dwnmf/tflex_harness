# Agent Workspace Snippets

Temporary C# snippet candidates can be stored here for manual review before live verification.
Normal tool runs write reproducible copies under `artifacts/runs/`.

Use `python -m tflex_harness.cli save-snippet <name> --code "<C# code>"` or the MCP tool `save_tflex_snippet_candidate`.

Candidates are not verified recipes. Promote them to `agent_workspace/recipes` only after documentation evidence, compile diagnostics, live run evidence, and documented assumptions/limitations.
